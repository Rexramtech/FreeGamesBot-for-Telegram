# FreeGamesBot-for-Telegram
<h1>A Bot that informs you the new free games available for PC </h1>

The first step is to Install telegram to your device phone
Second , search BotFather, type in BotFather  the command /start , /newbot ,  Name your bot and the BotFather willl gives you a token to acces the HTTP API , keep your token secure and store it safely.
The next step is to Edit the document called docker-compose.yml in visual studio putting the token thats had givent to you by de BotFather, you must copy the token and paste next to BOT_TOKEN=
